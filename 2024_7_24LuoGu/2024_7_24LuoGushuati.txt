题目描述：
求两个整数的和。
输入格式：
一行，两个用空格隔开的整数。
输出格式：
两个整数的和，且对于 100% 的数据，输入的整数在[1,4×10^18]内。


#include <stdio.h>
int main()
{
 long long i=0;
 long long j=0;
 scanf("%lld %lld",&i,&j);
 long long a=i+j;
 printf("%lld",a);
 return 0;
}
